# Heads or Tails - Coin Toss Image Classification

This project is a **binary image classification** for the Kaggle "Heads or Tails?" competition.

## Features
- **Transfer Learning:** VGG16 (ImageNet weights)
- **Input:** Coin images (224x224)
- **Output:** Heads / Tails probability
- **Framework:** TensorFlow + Keras
- **Deployment:** Streamlit web app (Hugging Face)

## Usage

### Streamlit
```bash
streamlit run app.py
